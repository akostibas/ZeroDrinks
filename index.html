Detailed instructions

Adding to Home Screen (Apple's guide)

Long press the shortcut in the Shortcuts app, select "Details", then "Add to Home Screen". You can customize the icon and name.

Adding to Apple Watch (Apple's guide)

In the Watch app on your iPhone, scroll down to Shortcuts and enable "Show on Apple Watch". You can also add shortcuts to watch face complications.

Setting Up Automation for Zero Drinks (Apple's guide)

Important: Zero Drinks requires creating an automation to run daily. Here's how:

  1. Open the Shortcuts app and tap the "Automation" tab
  2. Tap "+" to create a new automation
  3. Choose "Time of Day"
  4. Select "Sunrise" and set to "1 hour after"
  5. Set to run daily
  6. Tap "Next" and search for the "Zero Drinks" shortcut
  7. Disable "Run Immediately" to keep confirmation enabled
  8. Tap "Done"

With confirmation enabled, you'll receive a notification each morning that you can tap to run the shortcut and see your stats.
